You should have Git installed at this point. If not, see Materials (Windows Users), Materials (Mac Users), or Materials (Linux Users). This means you already have Git ready to go. But how? We created a repo on GitHub but need a way to connect that repo with our personal computer. That's a great feature of Git: to connect the files on your computer to GitHub. It also performs version control and automatically records changes that have been committed.
