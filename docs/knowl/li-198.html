Item 2.

There is a dialog box indicated which branches are being merged. There are more boxes than before. There should be four, allowing you to specify which repo and branch to want to merge from and which repo and branch you want to merge to. Usually GitHub gets it right, but not always, so check to make sure these are what you want.

The base repository is the repo you are merging to (the authoritative repo), the head repository is the repo you are merging from (your fork). The base branch is the specific branch you are merging to (often main) and the compare branch is the branch from your fork you are merging from (often main for me personally).
