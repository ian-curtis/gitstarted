Section 1.4 Keyboard Shortcuts

You may be able to use your computer fine without any shortcuts. But wow, do they make your life easier. This section could go in an appendix but I think it is so important to know at many shortcuts as possible to help you use your computer faster. I use shortcuts every day in my web browser, programming software, documents editing software (e.g., Microsoft Word), and just my computer in general. I have tried to break up the following shortcuts by the keys needed. Ctrl + F works well on this page.

Note: Windows uses Control (or Ctrl) as the “main” button whereas Mac usually uses Command (or Cmd). I regularly use a Mac and Google Chrome so many of these might be more accurate and detailed for Chrome and Mac users.

Subsection 1.4.4 VS Code Shortcuts

This part lists many useful shortcuts with work in VS Code. Most should work in both Windows and Mac, but any differences are noted.

Base Key(s) Variable Key

Function

Notes
Alt + Click

Create multiple cursors (each click is one cursor)

Useful for changing multiple occurances at once
Alt + Shift Click and Drag

Create a vertical line of cursors
Ctrl + `

Open the terminal window

Macs should also use Ctrl
Ctrl or Cmd + Arrows

Move cursor to extreme edge

For instance, the up arrow will take you to the very top of the file and the left arrow will take you to the very left of the line
Ctrl or Cmd + Alt Left/Right arrows

Navigate between open files
Ctrl or Cmd + Alt Up/Down arrows

Create new cursor above/below the current cursor

Select content to extreme edge

Move cursor to beginning/end of each work
Alt + Up/Down arrows

Move current line up/down

Selects based on current position of cursor
