Creating a branch is not difficult. To do so, first make sure you are on the “Code” tab of your repository titled aboutme. Below the list of tabs, you should have a row of items. The first is a button with a branch-like icon with the word “main”. Next to that, there is the same branch-like icon with “1 branch” and then a tag-like icon with “0 tags” (Figure 3.3.2.1).  51  Now,

  1. Click on the word “main”. This will open a drop-down menu with all of your current branches (Figure 3.3.2.1). You probably only have one branch: the main branch.

  2. We will be adding a part to our README file: our favorite animal. Type animal in to the textbox that appears. Your cursor should navigate their automatically.

  3. Since we don't have a branch named animal, GitHub asks us if we want to create a new branch with that name. That is indeed what we want to do. Click on “Create branch: animal from 'main'”
