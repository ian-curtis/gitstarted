To install Homebrew, go back to your terminal that you opened when checking if Git was installed. Paste in

/bin/bash -c "$(curl -fsSL
https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

as one line (not two) and press Enter. Don't worry about what each part does (to be honest, I'm not entirely sure either). It is safe and trustworthy (I did it on my computer).
