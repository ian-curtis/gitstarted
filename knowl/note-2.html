Note 1.3.5. Spaces in File Names.

A space (or any whitespace character) should never appear in a file name. We will see later why this is the case. To save yourself some hassle in the long run, never create folders or files with spaces. In their place, use the hyphen (kebab-case) or underscore (snake_case, recommended). You could also choose to use camelCase, PascalCase, or UPPER_CASE_SNAKE_CASE. See a Most Common Programming Case Types blog post to get more details on these cases.
