Let's create a commit message for our changes on the animal branch. In the “Commit changes” block, there is a text box that is autofilled with “Update README.md”. This is the default and for some instances may be enough. But I have the tendency to write slightly more detailed messages for my commit and I recommend you do so as well. Instead, type “add favorite animal”. There is no need to add the file name as this message will only be tied to the file(s) you have edited. (Later on, we will see that specific commit messages are only associated with the files you have staged and committed but that's a topic for Chapter 4.)
