If you are already familiar with an editor or if you like a particular one, use it! However, I personally prefer Visual Studio Code 12  which can be downloaded at the previous link. Throughout this book, I will refer to VS Code with some guidelines on how to find certain features. Should you decide to use another editor, it is up to you to learn how to use it. If you are a beginner with Git and the command line, I would stick with VS Code although as you gain experience you might think about switching to Sublime Text.
