The key thing to note is that Mac and Windows are designed differently. Mac uses forward slashes to indicate a folder change and Windows uses backslashes (in Git Bash, Windows users can use forward slashes…one reason for sticking with Git Bash!). But their structures are different as well. Notice that Windows has “C:” at the beginning. This is the C drive and there are other drives. Other differences are beyond the scope of this book. A brief history on file paths is given by How-To Geek 26  and a rather hard-to-read explanation can be found on Wikipedia 27 .
